# Release Notes: v1.2.7

## Overview
Elevate visual aesthetics across core views. 
**Value:** Users experience a more refined, premium interface with improved readability and spatial organization.

## Fixes & Improvements

### Calendar View
- Improve contrast and rendering of quadrant background colors by adjusting opacity and blurs.
  **Value:** Prevent overlapping calendar grid lines from blending into foreground task descriptions, significantly enhancing readability.
- Enlarge hour indicators directly on the daily calendar grid and adjust z-index overlaps.
  **Value:** Deliver clearer readability and immediately recognizable time boundaries to users.
- Increase internal padding inside Unscheduled task blocks and structurally separate the Task Status to sit above the primary Title.
  **Value:** Establish cleaner visual hierarchy making unscheduled tasks faster to parse.
- Re-align Unscheduled calendar section headers with their neighboring download controls.
  **Value:** Produce a structurally balanced and pixel-perfect header aesthetic.

### Matrix Dashboard
- Reconstruct the entirety of the Eisenhower matrix dashboard to precisely mirror reference mockups using exact hex translations (`#0b0d16`, `#111522`, `#242940`).
  **Value:** Execute a dynamic, floating dark-mode card aesthetic that matches premium design targets perfectly.
- Re-apply heavy, solid neon outlines and inherited neon text-colors to the 4 quadrant categories and backlog arrays.
  **Value:** Maintain a strong visual rhythm that subconsciously separates priorities for the user without friction.

### Global UI / Navigation
- Remove the static native macOS window title bar constraints and override the default OS header with a transparent, merged window implementation.
  **Value:** Produce a deeply unified, immersive "space" application shell matching top-tier native applications.
- Rename the "Brain Dump" module and all referencing tutorial strings to "Brainstorm".
  **Value:** Provide clearer, more professional user messaging and product terminology.
- Minimize the overall expanded width of the application sidebar.
  **Value:** Optimize the UI spread and deliver more screen real estate to the primary task matrix.

## Build and Stability
- Revert core underlying desktop frameworks.
  **Value:** Guarantee smoother cross-platform binary compilation and superior desktop operational reliability heading into deployment.
- Iterate upon continuous integration (CI) workflows for pathing and dependency caching.
  **Value:** Decrease build failure rates and speed up automated deployment pipelines.
